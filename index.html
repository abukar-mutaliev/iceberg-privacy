1. Общие положения

Данная Политика определяет порядок обработки и защиты персональных данных пользователей мобильного приложения «Iceberg» (далее — «Приложение»). Используя Приложение, пользователь подтверждает своё согласие с условиями Политики.

2. Какие данные мы собираем

Мы собираем следующие категории персональных данных:

  • Данные, предоставленные пользователем: имя, адрес электронной почты, номер телефона, адрес доставки (город, район, улица, дом и т.д.), данные, указанные при регистрации, сообщения в чате, история заказов.
  • Технические данные: модель устройства, версия ОС, IP‑адрес, идентификаторы устройства (в т.ч. Advertising ID при наличии), cookie‑аналогичные технологии и данные об ошибках приложения.
  • Данные для пуш‑уведомлений: данные, обрабатываемые сервисом OneSignal для доставки уведомлений о статусе заказа и важных сообщениях.

3. Цели обработки персональных данных

Персональные данные используются для:

  • регистрации и управления аккаунтом;
  • оформления и выполнения заказов, доставки;
  • связи с пользователем (включая уведомления о статусе заказа);
  • обеспечения работы функционала приложения (включая чат);
  • аналитики и улучшения сервиса;
  • мер по обеспечению безопасности и предотвращения мошенничества;
  • технической поддержки.

4. Правовые основания обработки

Обработка персональных данных осуществляется на основании:

  • согласия пользователя;
  • исполнения договора (оказание услуг — оформление и доставка товара);
  • выполнения требований законодательства РФ;
  • законных интересов оператора (улучшение сервиса, безопасность).

5. Передача данных третьим лицам

Оператор может передавать персональные данные третьим лицам в следующих случаях:

  • курьерские службы для выполнения доставки;
  • OneSignal для отправки пуш‑уведомлений;
  • платёжные провайдеры, если они будут подключены для приёма оплат;
  • хостинг‑провайдеры и технические подрядчики;
  • по запросу уполномоченных государственных органов в соответствии с законом.

6. Хранение данных

Персональные данные хранятся в течение срока, необходимого для выполнения целей их обработки, либо до удаления аккаунта пользователем, если иное не требуется законом или договором.

7. Защита персональных данных

Оператор применяет правовые, организационные и технические меры для защиты данных: шифрование передачи (HTTPS), ограничение доступа, резервное копирование, мониторинг и др.

8. Права пользователя

Пользователь вправе:

  • получать информацию о своих персональных данных;
  • требовать уточнения, блокирования или удаления данных;
  • отозвать согласие на обработку данных;
  • обжаловать действия оператора в уполномоченные органы (включая Роскомнадзор).

Запросы направляйте по адресу: iceberg.ingushetia@gmail.com.

9. Использование Cookies и Advertising ID

В приложении могут использоваться cookie‑аналогичные механизмы и рекламный идентификатор Android Advertising ID (AD_ID) для работы аналитики и доставки уведомлений сервисом OneSignal. Если вы хотите отключить использование рекламного идентификатора, это можно сделать в настройках устройства.

10. Дети

Приложение не предназначено для детей младше 13 лет. Мы не сознательно собираем данные о детях.

11. Изменения политики

Политика может быть изменена. Обновлённая версия будет опубликована по этому же URL с указанием даты последнего обновления.
